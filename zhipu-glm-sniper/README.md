# 智谱 GLM Coding 抢购助手

这是一个用于智谱 GLM Coding 套餐页面的 Tampermonkey 辅助脚本。它可以在每天上午 10 点开售前提前运行，自动打开页面、监控套餐按钮状态，并在目标套餐可购买时自动点击购买按钮。

> 说明：脚本只辅助点击到购买/确认前流程，最终订单确认和支付仍需手动完成。

## 独立目录

本工具发布在 `zhipu-glm-sniper/` 子目录，不会覆盖仓库根目录的随机抽背网站。

访问地址：

```text
https://kemenriderzzz.github.io/zhipu-glm-sniper/
```

## 文件说明

| 文件 | 用途 |
| --- | --- |
| `index.html` | GitHub Pages 中文入口页 |
| `zhipu-sniper.user.js` | 主脚本，安装到 Tampermonkey 后在 GLM Coding 页面运行 |
| `auto-open.user.js` | 辅助脚本，负责每天提前打开抢购页面 |
| `使用说明.md` | 完整中文使用说明 |
| `快速配置指南.md` | 快速安装和配置步骤 |

## 默认配置

```text
每天开售时间：10:00:00
提前运行：3 分钟
目标套餐：Pro
计费周期：连续包月
自动点击：开启
繁忙重试：开启
```

## 安装

1. 安装浏览器扩展 Tampermonkey。
2. 新建脚本，复制 `zhipu-sniper.user.js` 的完整内容并保存。
3. 再新建一个脚本，复制 `auto-open.user.js` 的完整内容并保存。
4. 打开 `https://bigmodel.cn/glm-coding`，右上角出现“GLM 抢购”面板即表示主脚本已加载。

## 使用流程

1. 确保两个脚本都已安装并启用。
2. 提前登录智谱账号。
3. 在主脚本设置里确认目标套餐和计费周期。
4. 可以关闭抢购页面，保持浏览器和 Tampermonkey 可运行。
5. 每天约 `09:57:00`，辅助脚本会自动打开 `https://bigmodel.cn/glm-coding`。
6. 主脚本会提前开始监控。
7. 到 `10:00:00` 开售时，脚本会刷新页面并继续监控。
8. 检测到目标套餐可购买后，脚本会自动点击购买按钮。
9. 手动确认订单并支付。

## 注意事项

- 真实页面 DOM 如果发生变化，可能需要更新选择器。
- 浏览器可能会限制自动打开新标签页；如未自动打开，请检查 Tampermonkey 权限和浏览器弹窗拦截设置。
- 抢购高峰期受网络延迟、账号状态、服务端排队等因素影响，脚本不能保证一定成功。
- 请自行确认购买内容、价格和支付信息。
