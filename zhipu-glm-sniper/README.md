# 智谱 GLM Coding 抢购助手

这是一个用于智谱 GLM Coding 套餐页面的 Tampermonkey 辅助脚本。它可以在每天上午 10 点开售前提前运行，自动打开页面、监控套餐按钮状态，并在目标套餐可购买时自动点击购买按钮。

> 说明：脚本只辅助点击到购买/确认前流程，最终订单确认和支付仍需手动完成。

## 不会和原随机抽背页面混淆

本工具发布在独立目录：

```text
/zhipu-glm-sniper/
```

原来的随机抽背网站仍然使用仓库根目录的 `index.html`，本工具不会覆盖根目录页面，也不会修改原来的抽背文件。

## 访问地址

```text
https://kemenriderzzz.github.io/zhipu-glm-sniper/
```

## 安装文件

- 主脚本：`zhipu-sniper.user.js`
- 自动开页脚本：`auto-open.user.js`
- 快速指南：`快速配置指南.md`

## 默认配置

```text
每天开售时间：10:00:00
提前运行：3 分钟
目标套餐：Pro
计费周期：连续包月
自动点击：开启
繁忙重试：开启
```

## 使用流程

1. 安装 Tampermonkey 浏览器扩展。
2. 新建脚本，复制 `zhipu-sniper.user.js` 的完整内容并保存。
3. 再新建一个脚本，复制 `auto-open.user.js` 的完整内容并保存。
4. 打开 `https://bigmodel.cn/glm-coding`，确认右上角出现“GLM 抢购”面板。
5. 在设置里选择目标套餐和计费周期。
6. 每天约 `09:57:00` 自动打开页面并开始监控。
7. 到 `10:00:00` 开售时刷新并继续监控。
8. 检测到可购买后自动点击，随后需要你手动确认订单并支付。

## 注意事项

- 真实页面 DOM 如果发生变化，可能需要更新选择器。
- 浏览器可能会限制自动打开新标签页；如未自动打开，请检查 Tampermonkey 权限和浏览器弹窗拦截设置。
- 抢购高峰期受网络延迟、账号状态、服务端排队等因素影响，脚本不能保证一定成功。
- 请自行确认购买内容、价格和支付信息。
