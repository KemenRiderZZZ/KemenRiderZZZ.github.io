## 目录结构

| 路径 | 用途 |
| --- | --- |
| `index.html` | 根目录项目导航页 |
| `404.html` | 通用 404 导航页 |
| `.nojekyll` | 关闭 Jekyll 处理 |
| `README.md` | 仓库说明 |
| `仓库文件说明.md` | 本文件 |
| `xingsu-koujue-random/` | 刑诉口诀随机抽背项目 |
| `zhipu-glm-sniper/` | 智谱 GLM Coding 抢购助手项目 |

## 维护建议

- 更新随机抽背项目时，修改 `xingsu-koujue-random/index.html`。
- 更新智谱抢购助手时，修改 `zhipu-glm-sniper/` 目录内对应文件。
- 根目录不再放具体项目页面，只做分类导航。
