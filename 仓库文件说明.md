# 仓库文件说明

这个仓库用于发布“刑诉口诀随机抽背”网页。

## 推荐打开

`index.html` 是 GitHub Pages 的正式首页。朋友访问网站时会自动打开它：

```text
https://kemenriderzzz.github.io/
```

## 文件名对照

| 文件名 | 用途 | 是否建议改名 |
| --- | --- | --- |
| `index.html` | 网站首页，GitHub Pages 会自动读取 | 不建议改 |
| `刑诉口诀随机抽背.html` | 中文名跳转入口，方便识别和本地打开 | 可以保留 |
| `404.html` | 网站访问错误时的备用页面 | 不建议改 |
| `.nojekyll` | 关闭 Jekyll 处理，保证静态网页原样发布 | 不建议改 |
| `README.md` | GitHub 仓库首页说明 | 不建议改 |
| `仓库文件说明.md` | 中文文件说明 | 可以保留 |

## 维护建议

以后如果更新网页内容，只需要更新 `index.html`。`刑诉口诀随机抽背.html` 会自动跳转到首页，不需要同步复制。
